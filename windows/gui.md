## Keyboard Shortcuts
{id: keyboard-shortcuts}

* Windows (key)
* Windows+D - Display and hide desktop
* Windows+E - Open File Explorer
* Windows+L - Lock computer
* Windows+M - Minimize all windows-gui
* Windows+Shift+S - Screenshot
* Alt-TAB
